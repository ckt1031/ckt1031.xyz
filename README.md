## Get Started

[Node.js](https://nodejs.org) 18 (or latest LTS) and [Bun](https://bun.sh) (Version 1) required.

Install dependencies:

```bash
bun install
```

To start the development server:

```bash
bun run dev
```

Build the project:

```bash
bun run build
```
