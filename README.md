## Get Started

[Node.js](https://nodejs.org) 18 (or latest LTS) and [pnpm](https://pnpm.io) are required to build this project.

Install dependencies:

```bash
pnpm install
```

To start the development server:

```bash
pnpm run dev
```

Build the project:

```bash
pnpm run build
```
